# 001 storefront

e-commerce site built with React, TypeScript, Vite, and Tailwind CSS.

## Pages

- `/` — Product feed (shop)
- `/product/:id` — Individual product detail
- `/selection` — Cart / selected items
- `/checkout` — Checkout form

## Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router v6

## Getting Started

```sh
npm install
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) in your browser.

## Build

```sh
npm run build
npm run preview
```
